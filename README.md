## Описание
Приложение для восстановления контактов на телефоне путем импорта их из телеграма. 

<img src="https://github.com/user-attachments/assets/f373bf74-b944-42b2-9d88-abec7ded8479" width="300">

## Особенности
- Не нужно авторизоваться в телеграм аккаунт, полная автономная работа, ваши данные в безопасности.
- Импорт контактов из Telegram в контактную книгу.
- Автоматический парсинг контактов из JSON и HTML файла, экспортированного из десктоп-клиента Telegram.
- Конвертация контактов в формат vCard для удобной интеграции с вашим мобильным устройством.
- Простой и интуитивно понятный интерфейс для пользователя.

## Требования
**Android 5.0 (API level 21)**

## Установить
[Скачать APK](https://github.com/Zelimkhan-Magomadov/Telegram-Contacts/releases/download/v1.1/TelegramContacts.apk)

## Инструкция

### Обратите внимание, что все действия можно выполнять только из версии Telegram для компьютера!

### 1. Зайдите в настройки Telegram и выберите раздел "Продвинутые настройки".

<img src="https://i.ibb.co/9gm4bTw/image.png" width="300">

### 2. Пролистайте страницу вниз и выберите опцию "Экспортировать данные".

<img src="https://i.ibb.co/3BmthK1/image.png" width="300">

### 3. Оставьте выделенным только пункт "Список контактов".

<img src="https://i.ibb.co/8Y4fbV5/image.png" width="300">

### 4. Пролистайте страницу до самого низа, выберите одну из опций "Машиночитаемый JSON" или "Читаемый HTML" и снова нажмите "Экспортировать".

<img src="https://i.ibb.co/Qv8VgRL/image.png" width="300">

### 4.1 Если вы недавно вошли в аккаунт через приложение на компьютере, экспорт контактов может быть недоступен. В этом случае после неудачной попытки экспорта откройте чат Telegram на устройстве, где вы были авторизованы ранее, и нажмите кнопку «Разрешить». После этого экспорт данных станет доступен

<img src="https://github.com/user-attachments/assets/6c805946-7c52-4aee-895f-8ba1561f72c3" width="300">

### 5. Загрузите полученный файл с контактами на свое Android-устройство. Это можно сделать через Telegram, отправив файл в чат «Избранное» или любой другой чат, а затем скачать его на смартфон.

<img src="https://github.com/user-attachments/assets/f48c3798-5e24-4dbc-83b3-a5381e98fbec" width="300">

## 5.1 В случае если вы выбрали формат HTML, то нужный файл находится в папке "lists" и называется contacts.html

<img src="https://github.com/user-attachments/assets/535a2f42-dd31-4154-bb29-b95ac49994d1" width="300">

<img src="https://github.com/user-attachments/assets/a42c8076-0284-4595-8573-6a8251e3f09b" width="300">
   
### 6. В приложении **Telegram Contacts** выберите экспортированный файл контактов, нажав на кнопку «Выбрать файл». Если вы скачали файл через телеграм, то он будет расположен по такому пути.

<img src="https://github.com/user-attachments/assets/b1458f9d-ced8-44b1-85ae-de21c7dc514d" width="300">

### 7. Нажмите «Конвертировать», и появится кнопка, при нажатии на которую вы сможете сохранить все контакты в телефон.

<img src="https://github.com/user-attachments/assets/6528c24e-0847-47c3-aa43-c6fb6a7a017a" width="300">

### 8. Выберите ваше системное приложение по управлению контактами.

<img src="https://github.com/user-attachments/assets/e44ee730-1f2a-47a7-a399-d46bdc0b9e09" width="300">

### 9. Вы также можете поделиться файлом с восстановленными контактами. Например, вы можете восстановить контакты для пользователя iPhone, передать ему этот файл, и он сможет открыть его через встроенное системное приложение для контактов.
